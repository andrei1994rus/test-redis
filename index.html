You should to use (you may see pathes of every command):
  • /person/lrange/all -output all items of list
  • /person/llen -count of list's elements.
  • /person/lindex/:index -get element of list by inputted index.
  • /person/lset/:index/:elem -set list's element by required index and inputted element.
  • /person/lrem/:elem -remove required element.
  • /person/lpush/:elem -push element (or elements) into list.
